# conda
  在python使用環境中常常遇到不同套件版本間相容性問題  
  例：tensorflow, pytorch等深度學習庫所需要對應的ldnn庫版本不同  
  甚至可能會因為套件衝突導致使用不便  
  conda可建立多個虛擬環境達到環境間分離的作用  
  針對不同使用需求快速切換相對應環境  
  常見有：  
    Windows : Anaconda  
    Linux : Miniconda
